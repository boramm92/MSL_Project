API Guideline

1) REST API is available wherever you can send HTTP requests.

ID & Key

1) You need a unique ID and key for Minds Lab’s API services.

2) Subscribe ‘Business Plan’ or ‘Enterprise Plan’ on maum.ai platform. (https://maum.ai)

Lip Sync Avatar

A naturally speaking avatar is created following the inputted text.

Preparation

- Input 1: Sentence text you want to hear

  • Restriction 1) : Sentence must be within 1000 characters (Korean standard)
  • Restriction 2) : The longer the input sentence, the longer it takes to process the video.

- Input 2: Background image file

  • Supported files : .jpg, .png
  • Image capacity : 10MB or less
API Document ​Upload : Insert the text and background image to be spoken and receive a request key.

① Request

  • Method : POST
  • URL : https://api.maum.ai/lipsync/upload

② Request parameters

Name Description type
apiId Unique API ID. Request from is required for the ID. string
apiKey Unique API key. Request from is required for the key. string
text Sentence (*50 characters or less) file
image Background image file

③ Request example 

curl --location --request POST
'https://api.maum.ai/feat/getHairInfo' \
--header 'Content-Type: multipart/form-data' \
--form 'apiId= Own API Id' \
--form 'apiKey= Own API Key' \
--form 'text= sentence to be spoken by avatar'
--form 'image= background image file for avatar'

④ Request parameters

Response
Name Description type
message API operation list
payload result list
message: API operation
Name Description type
message String describing the request processing status (Success/Fail) string
status Status code for request processing status (0: Success) number
payload: result
Name Description type
requestKey Request key for avatar creation string

⑤ Response example 

{
"message": {
"message": "Success",
"status": 0
},
"payload": {
"requestKey": "request key"
}
}
