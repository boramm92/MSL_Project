Loitering Detection

By specifying an area, you will be notified when someone wanders the area for a long time.

Preparation

- Input: video file

  • File type : .mp4
  • Size : Under 50MB
API Document

① Request

  • Method : POST
  • URL : https://api.maum.ai/abnormal-behavior/loitering:detect

② Request parameters

Key Description type
apiId Unique API ID. Request from is required for the ID. string
apiKey Unique API key. Request from is required for the key. string
video type:file (.mp4) video file file
roiList area of interest information, Area information to detect loitering string

③ Request example 

url --location --request POST 'https://api.maum.ai/abnormal-behavior/loitering:detect' \
--header 'Content-Type: multipart/form-data' \
--form 'apiId= Own API Id'
--form 'apiKey= Own API Key'
--form 'video= Input video file'
--form 'roiList="{\"overall\":[0,130,1280,130,1280,720,0,720]}"'

④ Response example 

{
	"message": {
		"message": "Success",
		"status": 0
	},
	"payload": {
		"status": "loitering",
		"abnormalImg": "w7/DmMO/w6AAEEpGSUYAAQEAAAEAAQAAw7/..."
	}
}
