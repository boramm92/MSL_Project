고령자 소지품 검출 (AIR-ObjectDetection)

이미지 내 고령자 소지품(15종)을 검출할 수 있는 엔진입니다.

샘플 파일로 테스트 하기

내 파일로 해보기

  • * 이미지 지원가능 파일 확장자: jpg, bmp
  • * 검출 대상 : 컵, 펜, 모자, 핸드폰, 양말, 안경, 수건, 지팡이, 신문, 리모컨, 열쇠, 지갑, 담배갑, 약통, 약봉지

이미지 분석중

이미지를 분석하는 데 약간의 시간이 소요됩니다. (최대 5분)

입력 파일

결과 파일

해당 엔진은 대한민국 과학기술정보통신부가 지원하는 정부 R&D 과제 “고령 사회에 대응하기 위한 실환경 휴먼케어로봇 기술 개발(AIR)” 중 한국전자통신연구원(ETRI)의 연구과제 성과물 입니다.

API 공통 가이드

개발 환경 세팅

1) REST API는 HTTP 요청을 보낼 수 있는 환경이라면 어디에서든 이용할 수 있습니다.

2) 다양한 환경의 웹 서버에서 활용 가능합니다. (Python, Web, Javascript, Java)

키 발급

1) Mindslab에서 제공되는 API 서비스를 사용하기 위해서는 키를 선지급 받아야합니다.

2) 마음AI 플랫폼에서 Business Plan 이상 구독 신청을 하셔야합니다. (https://maum.ai)

3) [계정메뉴]에서 본인의 API ID 및 Key를 확인해주세요.

4) 발급받은 API ID 및 Key를 기억하여 아래 매뉴얼에 맞게 사용합니다.

※ Mindslab API를 위한 ID와 키는 외부 유출 및 공유를 금합니다.
