Voice Generation (Text-To-Speech)

MINDs lab Voice Generation API converts texts into natural human voice.

Preparation

① Input: Text(String) *Under 1000 characters

② Language: Choose one below

  • English - Selena (baseline_eng)
  • English - Brandon (eng_male1)
  • Korean Female - Calm (kor_female1)
  • Korean Female - Cheerful (kor_female2)
  • Korean Female - Natural (kor_female3)
  • Korean Female - Warm (kor_female4)
  • Korean Female - Eccentric (kor_female5)
  • Korean Female - Sweet (baseline_kor)
  • Korean Male - Genuine (kor_male1)
  • Korean Male - Confident (kor_male2)
  • Korean Male - Calm (kor_male3)
  • Korean Male - Friendly (kor_male4)
  • Korean kids - Boys(kor_kids_m)
  • Korean kids girls(kor_kids_f)
API Document

① Request

  • Method : POST
  • URL : https://api.maum.ai/tts/stream

② Request Parameters

Name Description type
apiId Unique API ID. Request from is required for the ID. string
apiKey Unique API key. Request from is required for the key. string
text Input string (*Under 1000 characters) string
voiceName Selected voice model string

③ Request example

		curl -X POST \
		  https://api.maum.ai/tts/stream \
		  -H 'Content-Type: application/json' \
		  -d '{
			"apiId": "(*Request for ID)",
			"apiKey": "(*Request for key)",
			"text": "Welcome to Minds Lab.",
			"voiceName": "baseline_eng"
		}'

④ Response example

audio/x-wav type chunked data (StreamBody)
