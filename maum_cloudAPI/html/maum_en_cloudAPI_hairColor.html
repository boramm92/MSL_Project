Hair Segmentation

Recognizes and displays the area of hair in the image while classifying the color.

Preparation

- Input: Image File

  • Supported files : .jpg
  • Image capacity : 3MB or less
  • Image size : 200x200 or higher
  • Recognition is possible only when the front face isrecognized.
  • The higher the image resolution, the higher the accuracy of recognition.
API Document

① Request

  • Method : POST
  • URL : https://api.maum.ai/feat/getHairInfo

② Request parameters

Name Description type
apiId Unique API ID. Request from is required for the ID. string
apiKey Unique API key. Request from is required for the key. string
scene_img type : Image File(.jpg) string

③ Request example 

curl --location --request POST
'https://api.maum.ai/feat/getHairInfo' \
--header 'Content-Type: multipart/form-data' \
--form 'apiId= Own API Id' \
--form 'apiKey= Own API Key' \
--form 'scene_img= Image file for hair recognition'

④ Response example 

{
"message": {
"message": "Success",
"status": 0
},
"payload": {
"result_img": "/9j/4AAQSkZJRgABAQAAAQABAAD/2wBDAAIBAQEBAQIBAQECAg...",
"hairInfo": "Black Hair"
}
