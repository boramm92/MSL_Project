  • BEST 상품이미지 음성인식 Speech-to-Text 음성을 텍스트로 변환하는 엔진으로, 다양한 학습모델을 활용할 수 있고 높은 인식률과 빠른 처리 속도를 제공합니다.

  • NEW 상품이미지 실신 감지 Fall Down Detection 특정 지역에서 실신하는 사람을 인식합니다.
  • NEW 상품이미지 침입 감지 Intrusion Detection 영역을 지정하여 해당 영역을 누군가가 침입할 시 알려줍니다.
